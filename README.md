# SCPI Language Support

Language support for SCPI (Standard Commands for Programmable Instruments) using Tree-sitter.

## Features

- Syntax highlighting for SCPI files
- Code completion
- Symbol navigation
- Diagnostics and error checking
- AI context support

## Requirements

- VS Code version 1.96.0 or higher

## Extension Settings

This extension contributes the following settings:

* `scpi.*`: SCPI language specific settings
